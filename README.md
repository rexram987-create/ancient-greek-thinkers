# אנשי הרוח הגדולים של יוון העתיקה
# Great Thinkers of Ancient Greece

אתר מוזיאוני־לימודי דו־לשוני בעברית ובאנגלית, המוקדש לדמויות מרכזיות מחיי הרוח של יוון העתיקה.

A static bilingual museum-style website about major figures from ancient Greek intellectual life.

---

## סקירה בעברית

האתר מיועד לפרסום באמצעות GitHub Pages וכולל כיום:

1. דף בית דו־לשוני בעברית ובאנגלית.
2. מבוא קצר לתרבות הרוח של יוון העתיקה.
3. מפה אינטראקטיבית סכמטית.
4. ציר זמן קצר וסקירה של תחומי הידע.
5. גלריה כללית.
6. דפי ביוגרפיה מפורטים למספר דמויות.
7. תבנית משותפת להוספת דמויות נוספות בעתיד.

4. דמוקריטוס

6. הרודוטוס
7. תוקידידס
8. סוקרטס
9. אפלטון

12. ארכימדס

דפים מלאים קיימים כיום עבור הומרוס, פיתגורס, הרקליטוס, דמוקריטוס, היפוקרטס, הרודוטוס, תוקידידס וסוקרטס.

### מבנה המאגר

- `index.html` — דף הבית הראשי.
- `styles.css` — עיצוב משותף לכל האתר.
- `gallery.html` ו־`gallery.css` — דף הגלריה והעיצוב הייעודי שלו.
- `people/` — דפי הביוגרפיה.
- `people/person.css` — עיצוב משותף לדפי הדמויות.
- `people/person.js` — התנהגות משותפת לחלונות וללשוניות בדפי הדמויות.
- `people/person-template.html` — תבנית להוספת דמות חדשה.
- `assets/` — תמונות ונכסים חזותיים מקומיים.

### שלבי הפיתוח הבאים

- השלמת דפים לאפלטון, אריסטו, אוקלידס, ארכימדס וסופוקלס.
- הוספת תמונות מקומיות נוספות של פסלים, דיוקנאות וכתבי יד.
- הוספת צורת השם ביוונית ואטימולוגיה לכל דמות.
- הוספת מפות של מקום הלידה ומוקדי הפעילות.
- הרחבת צירי הזמן.
- הוספת יצירות מרכזיות ורעיונות מפורסמים לכל דמות.
- הוספת ביבליוגרפיה והערות מקור לכל עמוד.

## English Overview

The site is designed for GitHub Pages and currently includes:

1. A bilingual Hebrew/English home page.
2. A short introduction to ancient Greek intellectual culture.
3. A schematic interactive map.
4. A short timeline and field overview.
5. A general gallery page.
6. Individual biography pages for several figures.
7. A reusable person-page template for future additions.

### Main figures represented

The home page lists 13 major figures:

1. Homer
2. Pythagoras
3. Heraclitus
4. Democritus
5. Hippocrates
6. Herodotus
7. Thucydides
8. Socrates
9. Plato
10. Aristotle
11. Euclid
12. Archimedes
13. Sophocles

Dedicated pages currently exist for Homer, Pythagoras, Heraclitus, Democritus, Hippocrates, Herodotus, Thucydides and Socrates.

### Fields represented

- Philosophy
- History
- Medicine
- Mathematics
- Science
- Literature
- Drama

### Project structure

- `index.html` — main home page.
- `styles.css` — shared site-wide styles.
- `gallery.html` and `gallery.css` — gallery page and gallery-specific styles.
- `people/` — biography pages.
- `people/person.css` — shared styles for biography pages.
- `people/person.js` — shared modal and tab behavior for biography pages.
- `people/person-template.html` — template for adding another figure.
- `assets/` — local images and visual assets.

### Suggested next development stages

- Dedicated pages for Plato, Aristotle, Euclid, Archimedes and Sophocles.
- More local portrait, statue and manuscript images.
- Greek name forms and name etymology for every figure.
- Birthplace and activity maps.
- More detailed chronological timelines.
- Major works and famous ideas for every figure.
- Bibliography and source notes for every page.

## GitHub Pages

To publish the site with GitHub Pages:

1. Go to **Settings → Pages**.
2. Choose the `main` branch and the root folder.
3. Save and wait for GitHub Pages to publish the website.
