## English Overview

The site is designed for GitHub Pages and currently includes:

1. A bilingual Hebrew/English home page.
2. A short introduction to ancient Greek intellectual culture.
3. A schematic interactive map.
4. A short timeline and field overview.
5. A general gallery page.
6. Individual biography pages for several figures.
7. A reusable person-page template for future additions.

### Main figures represented

The home page lists 13 major figures:

1. Homer
2. Pythagoras
3. Heraclitus
4. Democritus
5. Hippocrates
6. Herodotus
7. Thucydides
8. Socrates
9. Plato
10. Aristotle
11. Euclid
12. Archimedes
13. Sophocles

Dedicated pages currently exist for Homer, Pythagoras, Heraclitus, Democritus, Hippocrates, Herodotus, Thucydides and Socrates.

### Fields represented

- Philosophy
- History
- Medicine
- Mathematics
- Science
- Literature
- Drama

### Project structure

- `index.html` - main home page.
- `styles.css` - shared site-wide styles.
- `gallery.html` and `gallery.css` - gallery page and gallery-specific styles.
- `people/` - biography pages.
- `people/person.css` - shared styles for biography pages.
- `people/person.js` - shared modal and tab behavior for biography pages.
- `people/person-template.html` - template for adding another figure.
- `assets/` - local images and visual assets.

### Suggested next development stages

Future versions can add:

- Dedicated pages for Plato, Aristotle, Euclid, Archimedes and Sophocles.
- More local portrait, statue and manuscript images.
- Greek name forms and name etymology for every figure.
- Birthplace and activity maps.
- More detailed chronological timelines.
- Major works and famous ideas for every figure.
- Bibliography and source notes for every page.

## GitHub Pages

To publish the site with GitHub Pages:

1. Go to **Settings -> Pages**.
2. Choose the `main` branch and root folder.
3. Save and wait for GitHub Pages to publish the website.
