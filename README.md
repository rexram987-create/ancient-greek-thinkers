# Great Thinkers of Ancient Greece  
# אנשי הרוח הגדולים של יוון העתיקה

A first foundation version for a historical website about the most famous and influential intellectual figures of ancient Greece.

גרסת יסוד ראשונה לאתר היסטורי על אנשי הרוח המפורסמים והמשפיעים ביותר של יוון העתיקה.

---

## English Overview

This project presents a digital museum-style website about major figures from ancient Greek intellectual life.

The first version includes:

1. A main index page.
2. A general English introduction.
3. A Hebrew introduction.
4. A card for each major figure.
5. A simple responsive design suitable for GitHub Pages.
6. A structure that can later be expanded into separate biography pages.

### Main figures included

1. Homer  
2. Pythagoras  
3. Heraclitus  
4. Democritus  
5. Hippocrates  
6. Herodotus  
7. Thucydides  
8. Socrates  
9. Plato  
10. Aristotle  
11. Euclid  
12. Archimedes  
13. Sophocles  

### Fields represented

- Philosophy
- History
- Medicine
- Mathematics
- Science
- Literature
- Drama

### Suggested next development stages

Future versions can add:

- A separate page for each thinker.
- Portraits, statues and manuscript images.
- Hebrew and English names.
- Greek name forms.
- Etymology of each name.
- Birthplace maps.
- Chronological timelines.
- Major works.
- Famous ideas.

## GitHub Pages

To publish the site with GitHub Pages:

1. Create a new repository, for example: `ancient-greek-thinkers`.
2. Upload `index.html`, `styles.css`, and `README.md`.
3. Go to **Settings → Pages**.
4. Choose the `main` branch and root folder.
5. Save and wait for GitHub Pages to publish the website.
